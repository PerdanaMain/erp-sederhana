# Project Description

Project ini merupakan aplikasi ERP sederhana

## Project Specification

-   Laravel 10
-   PHP 8
-   MySQL 8

## How to Install

1. Clone this repository
2. Set up your environment
3. Run `composer install`
4. Run `php artisan migrate --seed`
5. Run `php artisan serve`
